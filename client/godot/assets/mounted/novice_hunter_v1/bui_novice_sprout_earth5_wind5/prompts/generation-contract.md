# AI 整图生成合同

本包于 2026-07-18 使用 OpenAI 内置图像生成能力制作。输入只包含本项目原创的见习猎人、芽耳布伊身份参考和已经通过自评的完整骑乘方向稿；未使用第三方游戏贴图。

生成要求：

1. 每张输出从头到脚完整重绘人物、宠物与鞍具，三者是一个连续主体，不输出可拆分人物层或宠物层。
2. 八个方向分别生成。镜头为高清 2.5D 等距游戏精灵视角，透明化前使用单色背景，不带投影、文字、界面或地面。
3. 人物始终按正常成年见习猎人比例跨坐，骨盆落鞍、双腿分列两侧，人物与宠物朝向一致。
4. 每方向四张行走相位必须保持身份和轮廓稳定，表现四足交替与骑手随鞍的轻微起伏，不用整体平移冒充动作。
5. 生成后仅对完整主体执行色键清理、整体裁切、整体等比缩放与统一脚底基线；不得在像素层把人物和宠物分别加工后再组合。

验收要求：八方向各自独立、40 帧可读、运行时不镜像、骑乘身体只绘制一张纹理；通用地面阴影可以由引擎单独绘制。
