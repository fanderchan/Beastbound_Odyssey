
# integrated rider+tiger world generation contract

- Eight independently authored directions: south, southwest, west, northwest, north, northeast, east, southeast.
- Each direction contains one idle frame and four explicit walk phases; no runtime mirroring.
- Exact orange-and-cream juvenile tiger identity, stable stripes, round ears, broad paws, clean 2.5D sprite silhouette.
- Rider and tiger are generated as one whole image in every frame; adult rider scale, stable saddle contact, no layered composition.
- Flat `#FF00FF` generation background; runtime extraction is 256×256 RGBA with at least 4px alpha safety margin.
