
# 见习猎人骑岚纹幼兽身份锁 v1

状态：`self_review_passed_owner_pending`。本锁只批准前/背三分之四整体骑乘关键姿势；不批准世界真八向或骑乘战斗动作。

## 整体生成合同

- 两姿都是 AI 一次生成的“见习猎人 + 岚纹幼兽 + 低鞍 + 短缰”完整主体，不是人物层和宠物层拼贴。
- `runtimeLayeredComposition=false`、`runtimeMirroring=false`；不得靠遮罩补缝、矩形前景片或镜像修复。
- 前姿人宠共同朝左下，背姿人宠共同朝右上。

## 人物、坐骑与受力

- 固定成年见习猎人身份与正常成人比例；人物不得缩成儿童、玩偶或大头吉祥物。
- 通过把岚纹幼兽整体放大到可信的结实矮马/熊级幻想坐骑尺度承载人物，仍保持原幼兽脸型和物种身份。
- 骑手骨盆落在肩斑后方低鞍，双腿自然跨坐，双靴完整并避开四足；双手握短缰，人物与坐骑共享运动轴。
- 坐骑身份固定：浅金奶油厚毛、单枚后倾青玉短角、每耳一枚贴附羽毛尖、每侧三层青玉肩羽、略长腿、三爪厚掌和实体分叉尾簇；鞍、缰和人物不能遮住短角、眼、肩斑或尾部剪影。

## 禁止漂移

- 人宠反向、人物浮空、人物坐在头颈、高背王座、肢体融合、手脚消失、坐骑物种改变。
- 后续动作不得长出翅膀或漂浮羽毛；耳尖、三层肩羽和分叉尾数量必须稳定。
- 不增加盔甲、武器、场景或元素特效；后续八向必须独立整图生成。
