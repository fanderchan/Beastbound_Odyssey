# 芽耳布伊动作生成记录

## 共用身份约束

- 以 `identity/identity-lock.md` 为唯一身份基准，不逐帧重新设计。
- 小型四足幼兽，头身约 1:1；沙黄色短绒毛，橄榄绿色背纹，奶油色心形口鼻与胸口。
- 两只巨大圆形绒耳，每只耳根只有一簇两片小嫩叶；四只泥棕色短爪；一条卷曲根须尾，尾端只有一片叶。
- 禁止兔、猫、狐、猪、人形、长耳叶片、翅膀、角、衣服、武器、文字、特效遮挡。
- 固定两套正式视角：`front_3quarter_sw` 与 `back_3quarter_ne`。所有帧使用相同轮廓、比例、配色、光照和透明底。
- 每张生成稿为严格等距网格、纯 `#FF00FF` 色键背景；母版后处理统一为 512×512 透明帧、共同比例、脚底对齐；Godot 运行帧再以 Lanczos 缩至 256×256，覆盖当前最大约 156 像素的实机绘制尺寸。
