# Phase43：宠物图鉴化详情

这一阶段把宠物面板里的详情拆成 `个体` 和 `图鉴` 两种视角。

## 规则

- 宠物面板详情区新增 `个体` / `图鉴` 切换。
- 默认进入 `个体`，保留原来的当前宠物详情。
- `个体` 显示当前这只宠物的等级、状态、生命、经验、能力值、技能。
- `图鉴` 显示形态模板资料，不修改当前宠物实例。
- 成长倾向只显示在 `图鉴`，不再混进默认个体详情。
- 图鉴只显示玩家可理解的中文，不显示 `formId`、`growthProfileId` 等内部字段。
- 底部操作按钮保持固定位置，详情文字多了仍然走滚动区。

## 图鉴内容

- 种系、亚种、形态。
- 属性配比。
- 成长倾向。
- 基础能力。
- 捕捉信息。
- 种系说明。
- 可用技能。
- 被动技能说明。

成长倾向来自形态模板的 `growthProfileId`，显示时映射为中文标签。后续如果出现 `5火5风` 这类混合属性，属性配比仍然独立显示为 `5火 5风`；成长倾向由模板字段决定，可以显示为 `攻击 / 敏捷` 这类组合。

## 自测

```sh
godot --headless --path client/godot --scene res://scenes/Main.tscn --quit-after 1800 -- --auto-pet-codex-detail-check
```

覆盖内容：

- `个体` / `图鉴` 按钮存在并能切换。
- 默认个体详情不显示成长倾向。
- 图鉴详情显示种系、亚种、形态、属性、成长倾向、基础能力、捕捉、技能、被动。
- 图鉴详情不显示内部 ID。
- 切换详情模式不会挤动底部操作按钮。
- 混合成长标签可以显示为 `攻击 / 敏捷`。
