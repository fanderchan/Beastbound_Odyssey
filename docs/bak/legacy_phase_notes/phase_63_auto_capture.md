# Phase63 自动捉宠与 GM 图鉴草丛

## 内容

- 内挂设置新增「捕捉」页签：自动捉宠开关、目标范围、图鉴下拉、手动输入、血量阈值、等级符号与等级值、捕捉工具优先级、无匹配时策略、捕捉时宠物技能、低战力自动丢弃。
- 无匹配时策略默认「逃跑」：一场战斗里没有任何一只怪满足捕捉条件时自动逃跑；需要压血或练级时可改为「战斗」。
- 捕捉时宠物技能默认「技2 防御」，避免宠物在玩家捕捉时把目标打死；后续可改成其他宠物技能。
- 自动捉宠触发捕捉时，同队陪练和陪练宠物本回合防御，避免队友误伤正在捕捉的目标。
- 若自动捉宠已有符合条件目标，但人物因低血先执行加血，本回合仍保留捕捉保护：宠物按捕捉页技能行动，陪练单位防御，不回落到普通自动攻击。
- 捕捉道具新增 `capturePower` 权重：空手 1、初级绳 3、捕捉网 6、强化网 10。自动捕捉会从玩家选择的工具向低权重工具降级兜底。
- 战力公式采用：`round(maxHp / 4 + attack + defense + agility)`；代码中 `quick` 作为 `agility` 的同义字段。
- 捕捉成功后仍按队伍优先、队伍满进兽栏；队伍和兽栏都满时提示「兽栏和宠物栏满，请清理」，不做交换逻辑。
- GM 练级测试场保留原 10V10 草丛，并新增「GM图鉴捕捉草丛」：随机出现 1 到 5 只任意可捕捉图鉴宠物，每只独立随机等级 1 到 10，少量敌人仍占用 10V10 的 1-10 固定号位。

## 自测命令

```bash
godot --headless --path client/godot --scene res://scenes/Main.tscn --quit-after 1800 -- --auto-capture-settings-check
godot --headless --path client/godot --scene res://scenes/Main.tscn --quit-after 1800 -- --auto-gm-10v10-map-check
```

## 手测入口

```bash
godot --path client/godot --scene res://scenes/Main.tscn -- --gm-10v10-map
```

打开完整客户端后，右下角「内挂」进入设置，切到「捕捉」页签；GM 地图里左侧大草丛仍是 10V10 测试，右上小草丛是随机图鉴捉宠测试。测试只抓 Lv1 时保留 `= Lv1`，没有 Lv1 会按「无匹配时」自动逃跑；想抓 1-10 级时可临时设为 `< Lv999`。
