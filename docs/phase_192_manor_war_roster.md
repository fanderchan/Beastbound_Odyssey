# Phase192：庄园战参战名单

本阶段把庄园战从族长单点结算推进到“成员报名参战”的家族战骨架。

## 已实现

- 宣战成功后，族长自动进入挑战方参战名单。
- 参战家族成员可在家族面板点击 `参战` 加入当前庄园战。
- 已参战成员可点击 `退出` 离开名单；挑战方最后一名参战成员不能退出。
- 结算时只计算参战名单成员战力，最多计入每方 5 人。
- 防守家族若无人参战，庄园仍保留基础守备战力；有人参战时取参战战力和守备底线的较高值。
- 玩家离开家族时会从进行中的庄园战名单移除；家族解散会取消相关进行中战期。

## 暂不做

- 庄园战专用多人 battle room。
- 战前入场地图、观战、踢人、锁名单。
- 按真实时间自动开战/超时结算。
- 胜负动画、战斗事件列表、庄园地图 NPC。

## 设计备注

- 当前上限 5 人是为了匹配现有可控的战力计算与 PC 面板展示；后续接入 N vs N battle room 后可以把名单扩展到真实战斗槽位。
- 商店权限仍只看最终占领状态，参战名单和宣战本身不会提前开放道具场。
