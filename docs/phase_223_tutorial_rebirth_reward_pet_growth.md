# Phase 223：教学宠分离与转生试炼/稀有奖励宠成长接档

日期：2026-07-11

## 范围与产品决定

P0.2d-2b 原本只计划为四只转生试炼兽和两只人物转生奖励幼兽接入服务端权威成长。实施前复现出一个产品冲突：`rebirth_starter_four_spirit_cub` 同时被新手教学蛋和人物第 5 转奖励使用，导致“所有新玩家免费宠”与“多次转生后罕见强宠”是同一只宠物。

本阶段按用户决定拆开两条价值线：

- 所有新玩家在“认识兽栏管理员”任务免费获得绑定的“芽耳布伊”教学蛋；芽耳布伊是较弱、不可捕捉、只带攻击与防御基础指令的入门宠。
- 四灵幼兽不再由教学蛋产出，继续作为人物第 5 转的罕见强宠奖励；第 6 转玄影幼兽保持更高一档。
- 四只 Lv50 转生试炼兽是人物转生所需的挑战/捕捉材料，不承担终局宠定位。
- 本阶段不制作宠物造型、动画或音频；芽耳布伊只登记原创资产需求与替换路径。

临时 Pet Design Contract 位于 `.run/pet-design/novice_sprout_bui_v1.json` 和 `.run/pet-design/p0_2d_rebirth_reward_growth_backfill.json`，均已通过项目宠物设计 Skill 校验，不提交版本库。

## 教学宠契约

新增形态 `bui_novice_sprout_earth5_wind5`：

- 名称：芽耳布伊；地 5 / 风 5；布伊系教学亚型。
- Lv1 基础：生命 80、攻击 11、防御 9、敏捷 46。
- 每级中心成长：生命 7.0、攻击 1.35、防御 1.10、敏捷 1.35。
- `weighted_center + 2% extreme` 个体分布，保留 Lv1 4V 和隐藏逐级成长的随机惊喜，但整体终局明显低于五转奖励宠。
- 绑定、不可野外捕捉、不能自动丢弃；当前只提供 `pet_attack` 与 `pet_defend`，不提前赠送构筑技能。
- 原 `novice_battle_pet_egg` 稳定 ID 不变，只把孵化形态改为芽耳布伊，因此任务、背包与旧接口无需破坏性迁移。

旧账号兼容：已有四灵幼兽仍被“设置战斗宠物”教学目标接受，也会阻止重复补领教学蛋。既有四灵幼兽、旧教学进度和已保存任务不删除、不改属性、不重抽成长。尚未孵化的旧 `novice_battle_pet_egg` 按当前道具契约孵化芽耳布伊；人物五转专用的 `rebirth_starter_four_spirit_cub_egg` 始终孵化四灵幼兽。

## 试炼兽数据语义修正

四只试炼兽模板此前把 Lv50 战斗展示值直接写进 `baseStats`。当其接入 authority-v1 后，这些值会被再次成长 49 级，产生远超设计的双重叠加。本阶段将模板重建为真实 Lv1 底板，并用每级成长在 Lv50 中心值回到原战斗基准：

| 形态 | Lv1 生命/攻/防/敏 | 每级生命/攻/防/敏 | Lv50 中心值 |
| --- | --- | --- | --- |
| 地之试炼兽 | 89 / 15 / 11 / 11 | 8.8 / 1.25 / 1.65 / 0.75 | 520 / 76 / 92 / 48 |
| 水之试炼兽 | 84 / 13 / 14 / 15 | 8.5 / 1.20 / 1.30 / 1.05 | 501 / 72 / 78 / 66 |
| 火之试炼兽 | 88 / 15 / 17 / 16 | 8.0 / 1.65 / 1.05 / 1.10 | 480 / 96 / 68 / 70 |
| 风之试炼兽 | 88 / 17 / 17 / 19 | 7.8 / 1.25 / 0.95 / 1.90 | 470 / 78 / 64 / 112 |

这只改变以后新生成的试炼兽个体。没有 authority envelope 的旧实例仍按 `legacy_existing` 保留现值，不把 Lv50 旧宠倒推、降级或重抽。

## 万人模拟与奖励层级

七个新增成长档各模拟 10,000 个个体，从 Lv1 使用正式 Node authority 算法逐级结算到 Lv140，并写入 2–140 共 139 级观察阈值：

| 形态 | 万人 Lv140 战力范围 | 平均 |
| --- | ---: | ---: |
| 芽耳布伊 | 791–932 | 857.58 |
| 地之试炼兽 | 796–944 | 872.30 |
| 水之试炼兽 | 778–928 | 852.21 |
| 火之试炼兽 | 798–957 | 876.22 |
| 风之试炼兽 | 835–995 | 916.03 |
| 四灵幼兽（五转） | 1112–1339 | 1224.83 |
| 玄影幼兽（六转） | 1184–1456 | 1312.64 |

结果形成清楚的三层：免费教学宠与试炼材料在约 850–920，五转四灵幼兽跃升到约 1225，六转玄影幼兽约 1313。四灵幼兽不会再因免费发放稀释罕见度；玄影仍保持更高转生门槛对应的数值优势。最终平衡仍需真实 PvE/PvP 手感验证，本阶段不把万人战力当作唯一强度结论。

## 服务端权威与跨端兼容

- 新教学宠、四只试炼兽、四灵幼兽和玄影幼兽都新增版本化 `growthSpeciesProfileId`；新获得实例由 factory 直接生成 canonical authority-v1。
- 孵蛋、GM 创建、逐级 EXP、转生成长周期与公开观察均复用同一服务端成长目录；客户端不生成私有品质、不重掷。
- 任务 `legacyFormIds` 只用于兼容已有四灵幼兽完成旧教学，不改变五转奖励来源或四灵自身身份。
- 当前 21 个可捕捉形态已经全部接入 authority-v1；仅新手老虎和雷龙两只非捕捉坐骑仍未接档。
- 组队经验回归夹具原先让低敏 authority 宠随机抢不到任何击杀，完整套件会偶发假红；本阶段只降低其他夹具攻击者的敏捷，连续 20 次锁定 authority 宠确实获得经验，不改游戏战斗或经验规则。
