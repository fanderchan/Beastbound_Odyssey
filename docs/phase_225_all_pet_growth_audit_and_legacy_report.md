# Phase 225：全物种成长总审计与真实旧档只读报告

日期：2026-07-11

## 范围

P0.2d-3 关闭宠物成长接档阶段的最后三个问题：

1. 32 个正式形态是否都能用同一服务端 authority-v1 算法从 Lv1 逐级结算到 Lv140；
2. 玩家在约 Lv20 是否有足够的观察证据判断去留，同时不直接读取隐藏品质；
3. 现有旧档在模板全部接档后是否会被误判为“应该补种子并重抽”。

本阶段不修改真实玩家数据，不执行迁移，不把旧宠转换成 authority-v1，也不改变任何成长中心数值。

## 32 万个体总审计

使用 `tools/pet_growth_population_audit.mjs --all --samples 10000` 对 32 个成长档各生成 10,000 个私有个体，共 320,000 个；每个个体使用正式 Node authority 算法从 Lv1 逐级结算到 Lv140。

结果：

- 32/32 成长档通过 Lv1 血量范围、三项成长范围、生命成长范围和 Lv140 战力目标；
- 报告无错误，位于 `.run/godot/p0_2d_all_32x10000.json`；
- 正式目录为 32 个形态、32 个物种成长档、32 个已接档形态；
- 21 个可捕捉形态、教学宠、六只人物转生奖励宠、四只试炼材料宠与两只坐骑使用同一权威创建/升级边界。

本阶段只生成 `.run` 报告，不重写已嵌入的观察表；前两个切片已经分别用相同 10,000 样本写入新增档案。

## Lv20 观察证据

每个成长档都内置 2–140 共 139 级的战力成长分位表，样本数固定为 10,000。客户端对联网 authority 宠只使用以下公开事实：

- Lv1 四项；
- 当前四项；
- 已实际升级次数；
- 当前等级对应的万人分位阈值。

“预测140”继续按 Lv1→当前的实测平均成长线性外推，不读取 private seed、private roll 或未发生的未来随机结果。Lv1 显示待观察，Lv2 起逐步形成预测；练到约 Lv20 后噪声被 19 次升级摊薄。

32 个档案的 Lv20 P25→P85 战力成长间距：

- 最小 `0.210/级`；
- 最大 `0.474/级`；
- 平均 `0.367/级`。

全部阈值满足 `min ≤ P25 ≤ P55 ≤ P85 ≤ P95 ≤ max` 且 `P85 > P25`。这意味着明显低成长、常见中段和明显高成长有可分辨空间，但界面仍展示“观察得到的分位与外推”，不是直接公布隐藏底档。Node 回归测试现在逐档锁定 10,000 样本、Lv20 单调区间与有效宽度。

## 真实旧档只读审计工具

新增 `tools/pet_growth_legacy_save_audit.mjs`：

- 必须显式传入一个或多个 `--input`；目录只发现 `player_profile.json`，可选 `--include-last-good`；
- 跳过符号链接，不执行 normalize、RNG、成长初始化、升级、迁移或网络请求；
- 扫描 `petInstances`、兼容 `pets`、地面宠和训练伙伴宠物路径；
- 分类 authority private/public、已接档旧实例、仍未接档旧实例、未知形态和损坏 authority 声明；
- 对旧实例唯一建议是 `preserve_existing_without_reroll`；损坏或未知对象只进入人工复核，不自动修；
- 输入文件在扫描前后计算 SHA-256，报告 `inputFilesUnchanged`；输出路径若等于任一输入会直接拒绝；报告文件权限为 `0600`。

## 本机真实缓存结果

只读扫描本机 Godot `server_accounts` 下 42 份真实服务器缓存：

- 宠物引用：171；
- `legacy_existing_linked`：171；
- authority 损坏：0；
- 未知形态：0；
- 结构警告：0；
- 需要人工复核：0；
- JSON 解析错误：0；
- 输入文件哈希前后一致：`true`。

报告位于 `.run/godot/p0_2d_real_server_cache_legacy_audit.json`。这些是本机真实服务器缓存的兼容证据，不冒充一次 MySQL 写迁移；P0.5 仍负责版本化数据库迁移、备份、回滚和资产总量校验。

结果确认：模板后来接入 `growthSpeciesProfileId` 不会让无 envelope 的旧宠自动变成新宠。它们继续走 `legacy_existing`，不补种子、不重算 Lv1、不修改现值。只有接档后真正新创建的个体进入 authority-v1。
