# Phase39：自由存入 / 取出宠物

## 目标

这一阶段把兽栏从“只能看见仓库宠”推进到可操作的自由存取流程。

当前仍是开发期便利入口：玩家可以随时在宠物面板里存入和取出。后续兽栏可以移动到村里的 NPC 或建筑，但本阶段先把规则跑通。

## 规则

- 队伍宠物点击 `存入` 后进入 `storage`。
- 出战宠存入后不自动指定下一只出战宠，允许当前没有出战宠。
- 兽栏宠点击 `取出` 后进入 `standby`。
- 队伍未满时才能取出。
- 队伍已满时取出失败，只提示 `队伍已满。`
- 不做交换逻辑，不让玩家选择一只队伍宠替换进兽栏。

## UI

- 宠物面板底部保持固定按钮区。
- 状态循环按钮只显示 `战斗`、`待机`、`休息`。
- 兽栏按钮只显示 `存入`、`取出`。
- 宠物详情文字放在滚动区，文字变多不挤压底部按钮位置。

## 自动测试命令

```sh
godot --headless --path client/godot --scene res://scenes/Main.tscn --quit-after 1800 -- --auto-pet-stable-check
```

覆盖：

- 队伍宠可存入兽栏。
- 兽栏宠可取出到待机。
- 出战宠存入后不自动补另一只出战宠。
- 无出战宠时战斗不会生成我方宠物演员。
- 队伍满时取出失败，宠物仍留在兽栏。
