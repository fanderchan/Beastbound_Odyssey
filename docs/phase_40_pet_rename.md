# Phase40：宠物改名

## 目标

这一阶段给宠物管理面板加入改名能力。改名仍然只修改玩家持有的宠物实例，不改图鉴模板和种系数据。

## 规则

- 任意队伍宠或兽栏宠都可以改名。
- 名字保存到宠物实例的 `name` 字段。
- 名字会去掉首尾空白，并移除换行。
- 名字不能为空。
- 名字最多 8 个字。
- 改名不会改变宠物状态、属性、技能、经验或兽栏位置。
- 改名后的出战宠进入战斗时使用新名字。

## UI

- 宠物面板底部新增短按钮 `改名`。
- 点击后打开独立输入弹层，不挤压宠物详情和底部按钮区。
- 弹层按钮只显示 `取消`、`确定`。
- 校验失败时保留弹层，并在底部消息区显示原因。

## 自动测试命令

```sh
godot --headless --path client/godot --scene res://scenes/Main.tscn --quit-after 1800 -- --auto-pet-rename-check
```

覆盖：

- 改名按钮可见。
- 弹层能读取当前宠物名。
- 确认后详情和列表同步新名字。
- 空名字会被拒绝。
- 超长名字会被拒绝。
- 出战宠改名后，下一场战斗读取新名字。
