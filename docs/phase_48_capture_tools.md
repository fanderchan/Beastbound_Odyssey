# Phase48：捕捉方式 / 捕捉道具

这一阶段把 `捕捉` 从单一测试指令推进为可扩展的抓宠流程。这里不做宠物成长、资质、个体值或正式经济系统。

## 设计方案

- 玩家点 `捕捉` 后先进入捕捉方式菜单，再选择目标。
- `空手捕捉` 永远可用，不消耗道具，但成功率最低。
- 捕捉道具消耗玩家存档里的数量，失败也消耗。
- 初版捕捉方式：
  - `空手`：不消耗道具，成功率加成 `0%`。
  - `初级绳`：初级捕捉绳，成功率加成 `12%`，默认持有 `5`。
  - `捕捉网`：捕捉网，成功率加成 `22%`，默认持有 `3`。
  - `强化网`：强化捕捉网，成功率加成 `34%`，默认持有 `1`。
- 捕捉成功率由目标剩余生命、目标捕捉难度、捕捉方式和目标异常状态共同决定。
- 普通 UI 只显示 `机会：较低 / 一般 / 较高 / 很高`，不显示调试数字。
- 自动测试输出保留精确 `chance` 和 `roll`，方便复现。

## 概率规则

基础公式：

```text
chance = 0.42 - hpRatio * 0.22 - difficulty * 0.12 + toolBonus + statusBonus
```

- `hpRatio`：目标当前生命 / 最大生命。
- `difficulty`：模板里的 `capture.difficulty / 100`。
- `toolBonus`：捕捉方式加成。
- `statusBonus`：
  - 睡眠 `+18%`
  - 石化 `+16%`
  - 混乱 `+10%`
  - 中毒 `+5%`
- 最终成功率限制在 `5%` 到 `95%`。

## 手动自测

打开捕捉方式预览：

```bash
godot --path client/godot --scene res://scenes/Main.tscn -- --capture-tools-preview
```

进入后点 `捕捉`，应看到：

- `空手`
- `初级绳 x5`
- `捕捉网 x3`
- `强化网 x1`
- `返回`

选择捕捉方式后点野生宠物，战斗日志应显示对应方式，例如 `见习猎人 使用初级捕捉绳捕捉 野生乌力。`

## 自动回归

```bash
godot --headless --path client/godot --scene res://scenes/Main.tscn --quit-after 1800 -- --auto-capture-tools-check
```

这个检查覆盖：

- 空手捕捉不消耗道具。
- 初级捕捉绳失败也会消耗数量。
- 强化捕捉网在低生命目标上能成功。
- 睡眠状态会提高捕捉成功率。
- 捕捉菜单按钮显示道具数量，选择目标时显示机会档位。
